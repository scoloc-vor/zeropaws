# 🎮 弹弓大师 Pro - Slingshot Master Pro

一个无需服务器、通过广告变现的第一视角弹弓射击游戏。

## 📋 项目特性

### 核心玩法
- ✅ **第一视角射击** - 拉动弹弓瞄准射击
- ✅ **金币收益系统** - 击中目标赚取金币
- ✅ **体力机制** - 每局消耗1点体力（20点满）
- ✅ **装备升级** - 弹弓和子弹分级升级
- ✅ **关卡设计** - 递进式难度调整

### 变现系统
- 💰 **Banner 广告** - 屏幕底部常驻
- 📹 **激励视频** - 玩家主动观看获得奖励（最高收入）
- 📱 **插屏广告** - 关卡失败时展示
- 🎁 **金币不足提示** - 自然引导用户看视频

### 技术特点
- 🔧 **无需服务器** - 本地JSON存储
- 🎯 **离线可玩** - 完全离线游戏体验
- 💾 **自动存档** - 实时保存进度
- 🎨 **跨平台** - Unity支持iOS/Android/WebGL

## 🎮 游戏机制详解

### 弹弓射击
- 玩家在第一视角拉动弹弓
- 拉力越强 → 射程越远 → 威力越大
- 瞄准准星击中目标获得金币

### 金币收益
| 目标类型 | 基础奖励 | 完美击中 | 视频倍数 |
|---------|--------|--------|--------|
| 普通目标 | 10💰 | 15💰 | ×2 |
| 特殊目标 | 50💰 | 75💰 | ×2 |
| 奖励目标 | 200💰 | 300💰 | ×2 |
| 关卡通过 | 100💰 | 200💰 | ×2 |

### 体力系统
- 每局消耗 1 点体力
- 最大 20 点
- 自动恢复：每 5 分钟 +1 点
- 看视频：立即恢复满值

### 装备升级
```
弹弓升级 (Lv1→Lv100)
├─ 威力 +1%
├─ 准度 +0.5%
└─ 需花费金币逐级升级

子弹升级 (Lv1→Lv50)
├─ 伤害 +2%
├─ 速度 +1%
└─ 需花费金币逐级升级
```

## 🎯 关卡设计

### 关卡难度递进
```
Lv1-5：练习阶段
  ├─ 目标少，移动慢
  ├─ 金币高
  └─ 容易通过 → 培养习惯

Lv6-15：进阶阶段
  ├─ 目标增多，移动加速
  ├─ 难度适中
  └─ 需要一定技巧

Lv16+：高手挑战
  ├─ 目标复杂，移动极快
  ├─ 需多发弹射
  └─ 高奖励
```
